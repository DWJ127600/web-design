# Web Design 示例项目集

这是一个前端视觉与交互动效的练习项目，包含多个独立 Demo。每个子目录基本都可以直接打开 `index.html` 预览效果，适合学习页面动画、布局和交互实现。

## 项目特点

- 纯前端静态示例为主（HTML/CSS/JS）
- 各效果相互独立，便于单独学习与修改
- 包含图片、GIF、视频等素材用于展示动效

## 目录结构

- `文本遮罩/`：文本遮罩效果示例（含视频素材与样式文件）
- `水墨飞溅背景动画/`：水墨风背景动画效果
- `瀑布流布局/`：瀑布流图片布局示例
- `轮播图/`：多种轮播图实现与测试页面
- `页面滚动视差效果(1)/`：滚动视差效果示例一
- `页面滚动视差效果(2)/`：滚动视差效果示例二
- `鼠标移动响应效果/`：鼠标移动驱动的响应式视觉效果

## 如何使用

1. 进入任一示例目录。
2. 使用浏览器直接打开 `index.html`（或目录中对应的测试页面）。
3. 按需修改 HTML/CSS/JS 与素材文件进行二次开发。

## 说明

- 本仓库以练习与演示为目的。
- 部分目录含多份测试文件（如 `test.html`），可按实际需要选择入口。
